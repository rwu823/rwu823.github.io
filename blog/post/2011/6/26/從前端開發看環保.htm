指尖上的前端

從前端開發看環保

2011/6/26
LifeWork

曾經有朋友看到我的簡歷上寫著"愛環保"，提出了疑問說：「愛環保跟你的專業無關，為什麼要這麼寫？」所以很想寫下一篇關於自己的想法。

關於環保中的垃圾分類：
首先來談談Web標準吧， W3C 所推薦的 Web 標準裡面有一段是說：「把結構(Structure)、表現(Presentation)、行為(Behavior) 分離。」 當然這邊的結構指的就是(X)HTML、表現就是CSS、行為則是JavaScript，這三樣也是組成網頁前端最重要的三個元素。這句話意即該做什麼事的就讓它做什麼，而不是噁心的全部混雜在一起，所以就像環保中的垃圾分類一樣， 把相同的東西歸類在一起，這樣在資源回收與垃圾管理上，才能夠減輕處理者的負擔並得到最大的效益。

關於環保中的節能減碳：
在Web前端工程中有一大部分是在追求 Performance 的表現，這就意即有良好的效能可以大大降低 User CPU 的負擔， 進而達到減少二氧化碳排放的功效，節能減碳愛地球阿，看到這邊還能不為全天下偉大辛苦的前端開發者致敬嗎(自肥XD)。

寫在最後：
拿這兩件事來相提並論確實是偏執了，不過真的也希望喚起大家對環保與 Web 標準的重視，Web 髒掉了還不打緊，可是地球只有一個，由其最近日本大地震後所引起的各種天災， 覺得很感慨也蠻令人省思的，地球已經發出抗議的聲浪了，大家真的要好好愛護地球阿。
