Github 官方推薦的 Jekyll

jekyll 是 Github 官方推薦的靜態部落格編輯譯方式，這是 base on Ruby 的專案
所以運行時電腦要先安裝 Ruby
另外一個類似的是 Octopress 這是基於 jekyll 的
所以理論上二種應該大同小異
因個人對 Ruby 比較沒有深厚的感情，故 survey 時就略過這個方案
