[技巧] Style 你的 Chrome dev tool

2013/10/24
CSSdebugdev tool

先上效果圖:

因為自己平常寫 code 時慣用暗色系背景，有時突然跳到 debug tool 的白底時， 會讓眼睛產生強烈的反差而感覺不舒服

於是 google 了一下，果然強大的 chrome debug tool 還是有辦法改的，在這邊整理一下相關的資訊

chrome 設定檔位置

首先要確認自己的 chrome 設定檔位置，在網址列打about:version

Custom.css

所有的外觀其實是由 Custom.css 控制的，它的位置位於:
[your_chrome_profile_path]\User StyleSheets\Custom.css
如果沒有的話，自行創建一個即可
