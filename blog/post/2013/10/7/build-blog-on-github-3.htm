[教學] 用 NodeJS 編譯 Blog 到 Github (三)

2013/10/7
NodeJSJavascript

這篇會開始進入一些事前準備工作，會一步步教大家把需要的東西安裝好
看到這邊假定你已經有使用過 git 與 Javascript 基礎
如果先前已經有玩過 github 與 NodeJS 可以直接跳到第4步驟 :

1) 申請 Github 帳號

當然一開始最先要有的就是 Github 帳號
直接到官網申請就好，這邊就不多做贅敘了

2) 了解 Github Pages

Github Pages 其實就是 Github 上的 Host
要建立自己的 Github Pages 也相當簡單
只要在自己的帳號下創個 your_id.github.io 的 Repository 即可

3) 安裝 NodeJS

NodeJS download 到這邊可以選擇自己適合的環境，我這篇以 Window 為例
Window 上的安裝非常簡單，一直 下一步 就好
安裝完後我們可以開啟 cmd 測試一下是否成功: 

# 查詢 NodeJS 版本
$ node -v
v.0.10.20

# 查詢 npm 版本
$ npm -v
1.3.11

如果順利出現版本號，表示安裝成功

4) 安裝 NodeJS Module

這邊我們採用 Jade Stylus 來快速編寫
Jade
Stylus 

$ npm install jade
$ npm install stylus

等待 npm 提示安裝完成後，就可以用 JavaScript 編寫我們要的部落格了
